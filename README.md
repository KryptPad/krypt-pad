# Krypt Pad

Krypt Pad is a secure note and password manager using AES-256 bit encryption. Your data is stored in files on your devices.

## Project setup
```
npm install
```

### Compiles and watches files for changes
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
